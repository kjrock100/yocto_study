# 환경 변수로 내보내기 (Exporting Variables to the Environment)

## 4.2 개요

BitBake 레시피나 설정 파일에서 정의된 변수를, 실제로 실행되는 태스크(셸 스크립트 등)의 환경 변수로 전달하려면 `export` 키워드를 사용해야 합니다.

## 4.2.1 사용법

변수를 정의하고 `export` 지시어를 사용하여 내보냅니다.

```bitbake
export ENV_VARIABLE
ENV_VARIABLE = "value from bitbake"
```

또는 정의와 동시에 내보낼 수도 있습니다.

```bitbake
export ENV_VARIABLE = "variable-value"
```

이렇게 `export` 된 변수는 `do_compile`이나 `do_install`과 같은 태스크가 실행될 때, 셸 환경에서 `$ENV_VARIABLE` 형태로 접근하여 사용할 수 있습니다. `export` 되지 않은 변수는 BitBake 내부적으로만 사용되며, 생성된 셸 스크립트에는 포함되지 않습니다.
