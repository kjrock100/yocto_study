# bitbake-layers 스크립트를 사용한 레이어 추가 (Adding a Layer Using the bitbake-layers Script)

Yocto Project에서 레이어를 생성하거나 다운로드한 후, 빌드 시스템이 해당 레이어를 인식하고 사용할 수 있도록 설정해야 합니다. `bitbake-layers` 스크립트의 `add-layer` 서브 커맨드를 사용하면 이 과정을 쉽고 안전하게 수행할 수 있습니다.

## 1. 레이어 추가 명령어

`bitbake-layers add-layer` 명령어를 사용하여 `bblayers.conf` 파일에 레이어를 추가합니다.

```bash
bitbake-layers add-layer <layer-path>
```

**예시**:

```bash
bitbake-layers add-layer ../meta-custom
```

이 명령을 실행하면 지정된 경로의 레이어가 빌드 구성에 추가됩니다.

## 2. 동작 원리

이 명령어를 실행하면 빌드 디렉토리 내의 `conf/bblayers.conf` 파일이 자동으로 업데이트됩니다. 구체적으로는 `BBLAYERS` 변수에 해당 레이어의 절대 경로가 추가됩니다.

## 3. 이점

- **자동화**: 설정 파일을 직접 편집할 때 발생할 수 있는 문법 오류나 오타를 방지할 수 있습니다.
- **경로 처리**: 상대 경로를 입력하더라도 스크립트가 이를 올바른 절대 경로로 변환하여 설정 파일에 저장합니다.
- **즉시 적용**: 명령 실행 즉시 변경 사항이 적용되어 바로 빌드 작업을 수행할 수 있습니다.

## 4. 참고 사항
레이어를 추가하기 전에 해당 레이어가 현재 Yocto Project 버전과 호환되는지(`LAYERSERIES_COMPAT`) 확인하는 것이 좋습니다. 호환되지 않는 레이어를 추가하면 빌드 시 에러가 발생할 수 있습니다.
