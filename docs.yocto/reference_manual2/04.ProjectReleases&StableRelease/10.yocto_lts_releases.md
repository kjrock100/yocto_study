# Yocto Project 장기 지원 릴리스 (Long Term Support Releases)

Yocto Project의 **LTS (Long Term Support)** 릴리스는 제품 수명 주기가 긴 임베디드 시스템 개발을 지원하기 위해 설계된 특별한 릴리스 유형입니다.

## 1. 개요

일반적인 Yocto Project 릴리스(Stable Release)는 다음 메이저 릴리스가 나올 때까지만(약 7개월) 유지 관리되는 반면, LTS 릴리스는 **최소 4년** 동안 보안 패치와 버그 수정을 제공받습니다. 이는 자동차, 의료, 산업 자동화 등 장기간 유지 보수가 필요한 산업 분야에서 필수적입니다.

## 2. 주요 특징

*   **지원 기간:** 최소 4년 (커뮤니티 및 회원사의 지원에 따라 연장 가능).
*   **업데이트 범위:**
    *   보안 취약점 (CVE) 패치
    *   심각한 버그 수정
    *   빌드 인프라 수정
    *   *새로운 기능은 원칙적으로 추가되지 않아 안정성을 최우선으로 합니다.*
*   **주기:** 대략 2년마다 새로운 LTS 버전이 지정됩니다.

## 3. 주요 LTS 릴리스 목록

최근 지정된 주요 LTS 릴리스는 다음과 같습니다.

| 버전 | 코드명 (Codename) | 릴리스 날짜 | 비고 |
| :--- | :--- | :--- | :--- |
| 5.0 | **Scarthgap** | 2024년 4월 | 최신 LTS |
| 4.0 | **Kirkstone** | 2022년 5월 | 이전 LTS |
| 3.1 | **Dunfell** | 2020년 4월 | 구형 LTS |

## 4. 사용 권장 대상

*   제품의 수명이 1년 이상인 프로젝트
*   운영체제 업그레이드 비용을 최소화하고 안정성을 유지해야 하는 경우
*   지속적인 보안 업데이트가 필요한 상용 제품

---
*참고: LTS 정책은 Yocto Project 회원사들의 자금 지원과 커뮤니티의 기여로 유지됩니다.*
