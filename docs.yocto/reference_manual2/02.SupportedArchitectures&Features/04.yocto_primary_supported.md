# Yocto Project 기본 지원 (Primary Supported) 아키텍처

Yocto Project에서 **기본 지원(Primary Supported)** 아키텍처는 프로젝트의 핵심 품질 보증(QA) 프로세스의 일부로 관리되는 하드웨어 아키텍처를 의미합니다.

## 1. 정의

기본 지원 아키텍처는 Yocto Project 릴리스 엔지니어링 팀에 의해 지속적으로 빌드되고 테스트됩니다. 이는 해당 아키텍처가 Yocto Project의 릴리스 주기 동안 안정적으로 작동함을 보장한다는 것을 의미합니다.

## 2. 지원되는 주요 아키텍처

다음 아키텍처들은 `meta-yocto-bsp` 레이어와 `openembedded-core`를 통해 즉시 사용 가능합니다.

*   **ARM**
    *   ARMv7 (32-bit)
    *   ARMv8 (64-bit / AArch64)
*   **x86**
    *   x86 (32-bit)
    *   x86-64 (64-bit)
*   **MIPS**
    *   MIPS32
    *   MIPS64
*   **PowerPC**
    *   PPC32
    *   PPC64

## 3. 검증 및 테스트

이 아키텍처들의 안정성은 주로 **QEMU 에뮬레이터**를 통한 자동화된 테스트와 **참조 하드웨어**에서의 검증을 통해 확보됩니다. 개발자는 `qemux86`, `qemuarm` 등의 머신 정의를 사용하여 별도의 하드웨어 없이도 이미지를 빌드하고 테스트해 볼 수 있습니다.
