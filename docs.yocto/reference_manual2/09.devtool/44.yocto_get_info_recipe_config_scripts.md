# Yocto Project 레시피 구성 스크립트 정보 확인 (Get Information on Recipe Configuration Scripts)

Yocto Project에서 `devtool configure-help` 명령어를 사용하여 레시피의 구성 스크립트(configure script) 옵션을 확인하는 방법에 대해 설명합니다.

## 1. 개요

`devtool configure-help` 명령은 레시피의 빌드 구성 단계(`do_configure`)에서 사용할 수 있는 옵션들을 보여줍니다. 주로 Autotools를 사용하는 레시피에서 `./configure --help`를 실행한 것과 같은 결과를 제공하여, `EXTRA_OECONF` 변수 등을 통해 설정할 수 있는 옵션을 파악하는 데 도움을 줍니다.

## 2. 기본 사용법

```bash
devtool configure-help <recipe>
```

*   **`<recipe>`**: 구성 정보를 확인할 레시피의 이름입니다.

## 3. 사용 예시

`qemu` 레시피의 구성 옵션을 확인하고 싶다면 다음과 같이 실행합니다.

```bash
devtool configure-help qemu
```

출력 결과에는 `--enable-kvm`, `--disable-werror` 등 해당 소프트웨어를 컴파일할 때 선택할 수 있는 다양한 인자(Argument)들이 설명과 함께 표시됩니다.

## 4. 주요 용도

*   **레시피 커스터마이징:** 특정 기능을 활성화하거나 비활성화하기 위한 정확한 옵션 이름을 찾을 때 사용합니다.
*   **의존성 확인:** 구성 옵션을 통해 선택적 의존성(Optional Dependencies)을 파악할 수 있습니다.

---
*참고: 이 명령은 `configure` 스크립트가 있는 프로젝트(Autotools 등)에 유효하며, CMake나 다른 빌드 시스템을 사용하는 경우 지원 범위가 다를 수 있습니다.*
