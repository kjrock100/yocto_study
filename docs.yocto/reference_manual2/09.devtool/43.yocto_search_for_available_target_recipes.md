# Yocto Project 사용 가능한 타겟 레시피 검색 (Search for Available Target Recipes)

Yocto Project에서 `devtool search` 명령어를 사용하여 현재 설정된 레이어들 사이에서 사용 가능한 레시피를 검색하는 방법에 대해 설명합니다.

## 1. 개요

`devtool search` 명령은 현재 빌드 환경(`bblayers.conf`)에 포함된 모든 레이어를 대상으로 레시피를 검색합니다. 레시피의 이름, 요약(summary), 설명(description) 등을 기반으로 키워드 매칭을 수행합니다.

## 2. 기본 사용법

```bash
devtool search <keyword>
```

*   **`<keyword>`**: 검색할 키워드입니다. 레시피 이름이나 설명에 이 단어가 포함된 레시피를 찾습니다.

## 3. 사용 예시

`alsa`와 관련된 레시피를 찾고 싶다면 다음과 같이 실행합니다.

```bash
$ devtool search alsa
alsa-lib              1.2.4              ALSA sound library
alsa-plugins          1.2.2              ALSA Plugins
alsa-state            0.2.0              Alsa Scenario Files
alsa-tools            1.2.2              ALSA Tools
alsa-utils            1.2.4              ALSA sound utilities
...
```

출력 결과는 레시피 이름, 버전, 그리고 간단한 설명으로 구성됩니다.

## 4. 활용 팁

*   **정규 표현식:** 키워드 부분에 정규 표현식을 사용할 수 있어 더 정교한 검색이 가능합니다.
*   **레이어 확인:** 검색된 레시피가 어느 레이어에 있는지 확인하려면 `bitbake-layers show-recipes <recipe>` 명령을 추가로 사용할 수 있습니다.

---
*참고: `devtool search`는 타겟 장비용 레시피뿐만 아니라 네이티브(native) 및 SDK용 레시피도 검색 결과에 포함할 수 있습니다.*
